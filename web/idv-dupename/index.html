IdentityV Duplicate Name Generator

Create a duplicate name for IdentityV! This tool uses the invisible character "U+0E00" to create names that appear identical.

How to Use

Follow these steps to generate your duplicate name:

  1. Enter your name in the input field below.
  2. Select where you want to add the hidden character by clicking on the positions.
  3. Generate your duplicate name and copy it to your clipboard.

Or, copy that invisible character below and you make do it yourself:

Hint: If the game prompt the name is still duplicate, try adjusting the positions or the number of hidden characters.
